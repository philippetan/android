Privacy Policy for Pawfectly Adoptable

At Pawfectly Adoptable, we value the privacy of our users and are committed to protecting it. This Privacy Policy explains how we collect, use, and safeguard your information when you use our mobile application and related services. By using Pawfectly Adoptable, you agree to the practices outlined in this policy.

Information We Collect

1.1 Personal Information

We collect the following personal information to verify user identity and ensure the security of the adoption process:

  • Name
  • Email address
  • Phone number
  • Physical address
  • Photos of valid IDs and certificates
  • Information related to your role (Adopter or Shelter) and any details you choose to share in your profile (photo, location, etc.)

1.2 Non-Personal Information

Through Firebase, a service provided by Google, we automatically collect certain non-personal information to improve app performance and user experience:

  • Device information (e.g., model, operating system)
  • App usage data (e.g., session frequency and duration)
  • Location data (e.g., approximate city or region, derived from IP address)
  • Browser type and version (for web applications)
  • Aggregate data (e.g., total downloads, average session time)
  • Behavioral data (e.g., feature usage patterns)

Purpose of Data Collection

2.1 Personal Information

We collect personal information to verify user legitimacy, facilitate communication between users (for example, between adopters and rescue shelters), and ensure a safe environment for adopting animals.

2.2 Non-Personal Information

Firebase automatically collects non-personal data, which we use solely to analyze app performance, understand user behavior, and improve the overall experience for our users. Firebase’s data collection practices are governed by Google's Privacy Policy.

Data Storage and Retention

3.1 Storage Location

All data is stored on Firebase's Cloud Firestore, a secure cloud service provided by Google.

3.2 Data Retention

Data in Cloud Firestore is subject to the following retention policies:

  • Time-to-Live (TTL) Policies: We may set automatic expiration dates for certain documents to delete stale data within 24 hours after expiration.
  • Backups: Firestore supports scheduled backups with a retention period of up to 14 weeks.
  • Point-in-Time Recovery (PITR): PITR retains all data versions for up to seven days, enabling restoration to any point within that window.

Data Sharing

We do not currently share user data with third parties. We take reasonable measures to protect the information you provide from unauthorized access, use, or disclosure. However, no method of transmission over the Internet or electronic storage is 100% secure, and we cannot guarantee absolute security. If this changes in the future, we will update this Privacy Policy and notify users in advance.

User Rights and Controls

5.1 Access, Update, and Delete

Users have the right to access, update, or delete their personal information. They can do so directly through the app settings.

5.2 Opt-Out Options

Upon deletion, all associated information will be removed from our database, and the user will no longer have access to their account. However, certain information may be retained as required by law or for legitimate business purposes.

Cookies and Tracking Technologies

We use various tracking technologies to improve the app experience:

  • Cookies: Small files stored on users’ devices to remember preferences and track activity over time.
  • Web Beacons: Images or scripts that track interactions with content (primarily in emails and web pages).
  • SDKs (Software Development Kits): Firebase SDKs allow us to track app usage patterns and events.
  • App Instance IDs and Analytics Tags: These are used to monitor user interactions and optimize app performance.

Data Security Measures

We prioritize the security of your data and use Firebase’s security features to protect it. Key measures include:

  • Data Encryption: All data is encrypted in transit using HTTPS.
  • Authentication Controls: Firebase's secure authentication processes prevent unauthorized access.
  • Admin Access: Only authorized personnel have access to sensitive information, following strict access control policies.

Minors

Pawfectly Adoptable is intended for users over the age of 18. We verify age during the registration process by requiring valid government-issued IDs.

Changes to This Policy

We reserve the right to update this Privacy Policy. If changes are made, we will notify users by email at the address provided during registration. Users are encouraged to review the Privacy Policy periodically for updates.
